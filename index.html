How does this look like?


What this is.

  • This is a list of machine learning products that are making profit.
  • For each product, you can find the idea behind it. These are real world machine learning applications. And these are worth charging money for.
  • It includes their business model, e.g.: Subscription for SaaS, paid PRO features, paid APIs. Just a few of these products are free. I included them because they have future monetization plans.
  • It includes also some insights on the machine learning tech implemented. Such insights can be the starting point to begin learning more about their theoretical aspects.
  • Some of these products are the result of team effort from a big company or startup. Others are built by indie hackers.
  • The list is contains currently 100 products. It is is manually curated and growing. I am putting the list together by hand using sources like: Twitter, Indie Hackers, Product Hunt, machine learning and AI newsletters I am subscribed to, and more generally web search when I care about a particular technology.

Why this.

Why does this exist?

There are tons articles and blog posts that present cool machine learning tech. Though, not many machine learning techniques make it to production because they are too complex. Even fewer make it into a product that can actually charge money. This list is an effort to manually identify what are paid products that use machine learning as their core feature.

Who is this for?

This tool is for indie makers, entrepreneurs, and side hustlers who want to find machine learning product ideas to build. Companies could as well use this tool to identify if there are possible partners or competitors in their same space. It is also a tool for machine learning practitioners and researchers to identify real world problems to tackle.

What is this for?

This is a tool to exercise your idea muscle. It helped me to develop mine. You can find products ideas based on machine learning that: need improvement or can be applied to other scenarios. Also, you can see which business models is used. It can help choose the business model for your product. Or you might have a cheaper way to solve the same problem and compete on the price. You could also use these to learn cool machine learning techniques starting from their application and getting deep into their theory later on.
